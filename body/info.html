Detalhes sobre o desenvolvimento, metodologia e arquitetura do portal Consulta360.

Arquitetura

O Consulta360 é uma Single Page Application (SPA) modularizada em Vanilla JS. O ecossistema roda sobre Nginx utilizando roteamento via subdiretórios, sem uso de frameworks de renderização pesados no frontend, o que garante performance extrema.

  • Tailwind CSS (AntiGravity UI)
  • IndexedDB para Cache Local
  • Chart.js para visualização de dados
